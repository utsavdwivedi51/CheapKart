# CheapKart
A shopping kart to buy any product at cheap rate from other karts </br>
A comprehensive shopping kart with all major features. This is a full-stack single-file HTML application with localStorage as the database. </br> 
Live Demo: https://utsavdwivedi51.github.io/CheapKart/
Here's what's packed in:

**Auth System**
- Login / Register with localStorage database
- Session persistence across page reloads

**Home Page**
- Auto-playing hero banner slider with 4 slides
- Flash Sale with live countdown timer
- Best Sellers, New Arrivals, Trending Deals (horizontal scroll)
- 12-category icon grid
- Offers strip (Track Order, Best Offers, Cheapkart Plus, Alerts)

**Shopping**
- 24 real-world products across 12 categories (Mobiles, Electronics, Fashion, Appliances, Furniture, Beauty, Grocery, Books, Sports, Toys, Automotive)
- Category browsing with nav bar
- Product filters (price range, rating, discount)
- Sorting (Relevance, Price, Rating, Newest)
- Live search with dropdown suggestions

**Product Detail**
- Large image view with thumbnail gallery
- Ratings, reviews count, discount badge
- Available Offers section
- Pincode delivery check
- Specs / Reviews / Q&A tabs
- Add to Cart + Buy Now

**Cart & Checkout**
- Cart with quantity controls & remove
- Price breakdown with savings highlight
- 3-step checkout (Address → Summary → Payment)
- Multiple payment options (UPI, Card, Netbanking, EMI, COD)
- Order confirmation with Order ID

**Wishlist, Orders, Profile**
- Wishlist with Add-to-Cart from wishlist
- Orders history with Track/Return/Invoice/Review actions
- Profile with Personal Info, Addresses, Payment Methods, Notification Settings

**Extra Features**
- Cheapkart Plus page with tier info
- Notifications page
- Toast notifications throughout
- Responsive design (mobile-first)
- Login-gated features
